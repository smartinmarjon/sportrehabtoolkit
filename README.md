# Marjon Sport Rehabilitation Toolkit

Student-facing interactive learning platform for MSc Sport Rehabilitation students.

## Included

- Premium app-style homepage
- Section dashboards
- Clinical Assessment section
- Interactive Postural Assessment Checklist
- Shared CSS and JavaScript
- GitHub Pages-ready structure

## Publish on GitHub Pages

1. Upload all files and folders to your GitHub repository.
2. Go to Settings → Pages.
3. Choose Deploy from a branch.
4. Select main and /root.
5. Save.

The website will load from `index.html`.
